# STEM Reasoning Lab Frontend

Responsive Next.js and TypeScript learning workspace.

The interface includes:

- Mathematics, physics, and chemistry challenge selection.
- Custom problem entry.
- Student working area.
- Direct, guided, and self-consistency modes.
- Progressive hint disclosure.
- Verification, confidence, misconception, and follow-up panels.

Configure `NEXT_PUBLIC_API_BASE_URL` in `.env.local`, then run `npm run dev`.
